CDN via jsDeliver#

If you rather use a cached version of Mashiro, just add these code snippets under <head>.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/enstars/mashiro@{version}/mashiro.min.css"><script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script><script src="https://cdn.jsdelivr.net/gh/enstars/mashiro@{version}/index.min.js"></script>

{version} = Latest version, if possible. Could also use @latest
